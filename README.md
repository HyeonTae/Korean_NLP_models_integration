## Prerequisite
- 우리는 아래와 같이 실험을 위해 virtualenv를 권장합니다.
```     sh
    sudo apt-get install build-essential libssl-dev libffi-dev python-dev
    sudo apt install python3-pip
    sudo pip3 install virtualenv
    virtualenv -p python3 venv
    . venv/bin/activate
    # code your stuff
    deactivate
```
